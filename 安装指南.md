# 📱 钱迹 APP — 编译安装指南

> 跟着这份指南一步步操作，大约需要 **30-60 分钟**（大部分时间是下载）

---

## 第一步：下载并安装 Android Studio

1. 打开浏览器，访问：https://developer.android.google.cn/studio
2. 点击"下载 Android Studio"按钮（文件约 1GB）
3. 下载完成后，双击安装包，**一路点 Next** 即可
4. 安装完成后，首次启动会自动下载 Android SDK（约 2-3GB，需要等待）

> 💡 如果下载慢，可以挂梯子或找同事的 SDK 复制过来

---

## 第二步：打开项目

1. 打开 Android Studio
2. 点击顶部菜单：**File → Open**
3. 找到并选择这个文件夹：
   ```
   C:\Users\刘\WorkBuddy\20260406125040\MoneyTrace
   ```
4. 点击 **OK**
5. 等待 Android Studio 自动同步项目（右下角有进度条，需等待 2-10 分钟）

> 首次打开会下载 Gradle 依赖，需要网络，约 500MB

---

## 第三步：连接你的手机

### 手机端设置（只需做一次）

1. 打开手机 **设置**
2. 找到 **关于手机** → 找到 **版本号** / **MIUI版本** / **系统版本**
3. **连续点击 7 次**，出现"您已进入开发者模式"提示
4. 返回设置 → 找到 **开发者选项**（可能在"其他设置"里）
5. 开启 **USB 调试**

### 电脑端连接

1. 用 USB 数据线连接手机和电脑
2. 手机弹出提示："是否允许 USB 调试？" → 点击 **允许**（勾选"始终允许"）
3. 在 Android Studio 顶部工具栏，应该能看到你的手机名称

---

## 第四步：编译并安装到手机

1. 在 Android Studio 顶部工具栏，确认选中你的手机设备
2. 点击绿色的 **▶ Run 按钮**（或按 `Shift + F10`）
3. 等待编译（约 2-5 分钟），进度显示在底部 Build 窗口
4. 编译成功后，APP 会**自动安装并打开**在你的手机上 🎉

---

## 第五步：开启通知访问权限（核心步骤！）

APP 安装好后，**必须做这一步**，才能自动记账：

1. 打开手机 **设置**
2. 搜索"**通知管理**"或"**通知权限**"（不同手机叫法不同）
3. 找到"**通知使用权**"或"**已授权的应用**"
4. 找到"**钱迹**"，打开开关
5. 弹出提示时点击"**允许**"

### 各品牌手机的位置：

| 手机品牌 | 路径 |
|---------|------|
| 小米/MIUI | 设置 → 通知与状态栏 → 通知管理 → 通知使用权 |
| 华为/EMUI | 设置 → 应用 → 权限 → 通知访问权限 |
| OPPO/ColorOS | 设置 → 其他设置 → 辅助功能 → 通知监听 |
| vivo | 设置 → 应用 → 权限管理 → 通知类 |
| 三星/OneUI | 设置 → 通知 → 设备和应用通知 |

### 还需要关闭省电限制（防止被杀后台）

- **小米**：设置 → 应用 → 管理应用 → 找到"钱迹" → 省电策略 → 选"无限制"
- **华为**：设置 → 电池 → 应用启动 → 找到"钱迹" → 手动管理 → 全部开启
- **OPPO/vivo**：设置 → 电池 → 高耗电应用保护 → 添加"钱迹"

## ✅ 验证是否正常工作

1. 打开钱迹 APP，进入 **"我的"页面**
2. 顶部卡片显示 **"自动记账已开启 ✅"** = 成功！
3. 用手机打开微信或支付宝，**随便付一笔款**
4. 付款后几秒内，手机通知栏应该出现 **"💰 已自动记账"**
5. 打开钱迹，首页账单列表会看到这笔记录

## ❓ 常见问题

**Q: 提示"无法安装未知来源的应用"**
A: 手机设置 → 安全 → 允许安装未知来源应用（或安装未知应用）→ 允许 Android Studio

**Q: Android Studio 找不到手机**
A: 换一根 USB 线（有些线只能充电不能传数据）；或在手机上重新允许 USB 调试

**Q: Build 失败，提示 Gradle 错误**
A: 点击 Android Studio 菜单 → File → Invalidate Caches → Restart

**Q: 付款后没有自动记录**
A: 检查通知访问权限是否开启；检查后台是否被手机管家杀掉（设置自启动白名单）
